# multithreaded-text-editor
A Notepad-like text editor built with Python Tkinter, demonstrating core operating system concepts such as file I/O, process execution, and user interaction. Includes tabs, text formatting, autocomplete, code execution, and a drawing tool.

## 📝 Project Description

This project is a fully functional desktop text editor that mimics and extends the classic Windows Notepad. It includes advanced features such as multi-tab support, undo/redo, live status bar, Python code execution, and a built-in drawing tool.

This project was developed as part of the **Operating Systems** course to demonstrate GUI programming, file handling, process management, and event-driven architecture.

## ✨ Key Features

### Core Features
- **Multi-Tab Interface** – Work on multiple files simultaneously
- **File Operations** – New, Open, Save, Save As
- **Edit Tools** – Undo, Redo, Search, Replace
- **Formatting** – Change Font, Text Color, Background Color
- **View Options** – Zoom In/Out/Reset (30%–300%), Word Wrap

### Productivity Tools
- **Word Count & Statistics**
- **Auto Completion** (Python keywords)
- **Run Python Code** – Execute code directly from the editor
- **Built-in Drawing Tool** – Sketch and draw within the app

### User Experience
- Dynamic Status Bar (Line, Column, Zoom, Length)
- Keyboard Shortcuts
- Clean and intuitive UI


### Prerequisites
- Python 3.8 or higher installed
